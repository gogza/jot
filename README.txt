jot
    by Gordon McAllister
    http://github.com/gogza

== DESCRIPTION:

a little list making command line tool that uses Checkvist to store its lists

== FEATURES/PROBLEMS:

* configure jot to work with checkvist
* add items to a list at the command line

== SYNOPSIS:

  jot "Wash the car"
  jot "Mow the lawn"


== REQUIREMENTS:

* None so far!

== INSTALL:

* sudo gem install jot
